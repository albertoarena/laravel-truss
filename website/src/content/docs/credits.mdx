Laravel Truss stands on excellent open-source work.

## Built with

- **[Laravel](https://laravel.com/)**: the framework, and its native schema introspection (`Schema::getTables`, `getColumns`, `getIndexes`, `getForeignKeys`) that Truss reads the live structure through.
- **[Mermaid](https://mermaid.js.org/)**: renders the entity-relationship diagram in the browser. A copy is vendored and served from the package, so no CDN is required.
- **[IBM Plex Mono](https://github.com/IBM/plex)**: the monospace typeface of the dashboard, vendored under the SIL Open Font License.
- **[spatie/laravel-package-tools](https://github.com/spatie/laravel-package-tools)**: package scaffolding and service-provider conventions.
- **[Pest](https://pestphp.com/)**, **[Vitest](https://vitest.dev/)**, and **[Playwright](https://playwright.dev/)**: the PHP, unit, and browser test suites.
- **[Astro](https://astro.build/)** and **[Starlight](https://starlight.astro.build/)**: this documentation site.
